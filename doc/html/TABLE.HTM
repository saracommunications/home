TABLE element

Syntax:

<TABLE> ... </TABLE>

Example:

3 Column 2 Row Table with Border


<TABLE BORDER>
<TR> <TH>Col 1 Row 1</TH> <TH>Col 2 Row 1</TH><TH>Col 3 Row 1</TH></TR>
<TR> <TD>Col 1 Row 2</TD> <TD>Col 2 Row 2</TD><TD>Col 3 Row 2</TD></TR>
</TABLE>

Col 1 Row 1Col 2 Row 1Col 3 Row 1
Col 1 Row 2Col 2 Row 2Col 3 Row 2
