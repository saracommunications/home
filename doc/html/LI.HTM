LI Element

Syntax:

<LI> ... </LI>
NOTE: LI is always used inside either DIR, MENU, OL, or UL.

Example:


<UL>
<LI>This is the first item in the LIST.</LI>
<LI>This is the second.</LI>
<LI>This is the third.</LI>
</UL>
  • This is the first item in the LIST.
  • This is the second.
  • This is the third.
