HTML Element

The HTML element denotes a HTML document. It contains the actual HTML source. In order to maintain compatiblity with order browzers use of the HTML element is optional.

It can be in:

  • nothing

It can contain:

  • HEAD
  • BODY
  • FRAMESET

Syntax:

<HTML> ... </HTML>

Example:


<HTML>
<HEAD>
</HEAD>
<BODY> 
This is my HTML document.

</BODY>
</HTML>
