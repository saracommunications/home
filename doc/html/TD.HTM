TABLE DATA element

Syntax:

<TABLE><TR><TD> ...</TD> </TR></TABLE>
Note: The Table DATA element is always inside the TABLE Row element.

Example:


<TABLE BORDER>
<TR> <TD>Col 1 Row 1</TD> <TD>Col 2 Row 1</TD>/TR>
</TABLE>

Col 1 Row 1Col 2 Row 1

COLSPAN ATTRIBUTE EXAMPLE


<TABLE BORDER>
<TR>
<TD COLSPAN="2">This spans 2 columns</TD>
<TD>Column 3</TD>
</TR>
<TR>
<TD>Column 1</TD>
<TD>Column 2</TD>
<TD>Column 3</TD>
</TR> </TABLE>

This spans 2 columns Column 3
Column 1 Column 2 Column 3

ROWSPAN ATTRIBUTE EXAMPLE


<TABLE BORDER>
<TR>
<TD ROWSPAN="2">This spans 2 rows</TD>
<TD>Row 1, Col 2</TD>
<TD>Row 1, Col 3</TD>
</TR>
<TR>
<TD>Row 2, Col 2</TD>
<TD>Row 2, Col 3</TD>
</TR> </TABLE>

This spans 2 rows Row 1, Col 2 Row 1, Col 3
Row 2, Col 2 Row 2, Col 3
