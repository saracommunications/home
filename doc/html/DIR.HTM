DIR Element

Syntax:

<DIR> ... </DIR>

Example:


<DIR>
<li>This is the first item in the LIST.</li>
<li>This is the second.</li>
<li>This is the third.</li>
</DIR>
  • This is the first item in the LIST.
  • This is the second.
  • This is the third.
    •

    Many browsers, such as Netscape, present DIR lists the same as Unorder Lists.
