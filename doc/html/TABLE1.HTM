2 Column 3 Row Table with "5" wide Border

<TR> <TH>Col 1 Row 1</TH> <TH>Col 2 Row 1</TH></TR>
<TR> <TD>Col 1 Row 2</TD> <TD>Col 2 Row 2</TD></TR>
<TR> <TD>Col 1 Row 3</TD> <TD>Col 2 Row 3</TD></TR>
</TABLE>

Note: The BORDER=# attribute is supported by Netscape 2.0 and is included in the Proposed 3.2 HTML draft. No maximum is given. A border of 0 is not displayed.
